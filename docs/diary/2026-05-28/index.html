一、文章表格暗色模式修复

早上吴总反馈：文章《Agent时代，企业软件该"有头"还是"无头"？》中表格文字看不见。排查后发现是 global.css 中缺少 --color-bg-secondary 和 --color-bg-tertiary 两个 CSS 变量的暗色模式定义。暗色模式下回退到浅色背景值，浅色文字叠加在浅色背景上，自然看不见。

修复方案很简单：在 :root、prefers-color-scheme: dark 和 [data-theme="dark"] 三处同步补充变量定义。修复后线上验证，暗色模式表格背景色正确（--color-bg-secondary: #252333，--color-bg-tertiary: #2a2840），文字清晰可读。

这个小 Bug 带来了一条新规则：CSS 主题变量完整性检查。任何新增 var(--color-*) 变量时，必须在 :root 和暗色模式中同步定义，不能只加一处忘记另一处。这也是 lessons-learned 中的第 14 条规则。
